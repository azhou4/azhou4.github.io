Social Mobility in America: Data Visualization Project (December 2019)

As part of our final project for COMPSCI 171: Data Visualization, we illustrated the story of social mobility in America through a series of D3 visualizations using data from Opportunity Insights. View our project here!

Amazon Web Services (AWS) Documentation Website Feedback Form (August 2019)

Created and completed a customizable feedback service with AWS Lambda, S3, and API Gateway in combination with Python, HTML/CSS, JavaScript, React, and JSON to be used by all AWS products on the AWS documentation website in order to improve quality and usefulness of customer feedback. Before the project commenced, UI prototypes were first designed with Sketch.

"Growing Stars": JavaScript Background for Code.org's Dance Party (August 2019)

Created a fun animated background with p5.js for Code.org's annual Dance Party event. Code.org's mission is to give every child in the United States the chance to learn computer science in the classroom. This JavaScript background will be used for the 300 million students around the world next year for the 2020 Dance Party. The code can be found on my GitHub.

Practical Education Network: Teachers Guide (June 2019)

Through the MIT D-Lab: Education and Learning course, I worked closely with Practical Education Network (PEN), a nonprofit based in Accra, Ghana that works on implementing hands-on learning approaches in science and math classrooms. As a group of three, we dove deep into kinesthetic learning and project learning, and worked with PEN to develop teacher training materials that would aid in their teacher development workshops. In August 2019, I visited Accra for a week; I met with local teachers, toured schools, and learned more about the impact of PEN on local schools.

Women in Tech: Chrome Extension (December 2018)

Written in Javascript, the Women in Tech Chrome Extension was built for girls around the world to have role models in the technology industry. Every time a new tab is opened, a woman in tech appears on the page, along with a short biography, and links to learn more about her work. This Chrome Extension is live in the Chrome Store!

Thunderwear: iOS App (December 2017)

Thunderwear is an iOS app created using Swift, Python, and Google Maps API. Created as a final class project for CS50: Introduction to Computer Science, Thunderwear gives clothing item suggestions based on the weather/temperature of your location.
